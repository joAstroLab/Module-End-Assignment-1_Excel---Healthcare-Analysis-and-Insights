# Module End Excel Assignment - Healthcare Analysis and Insights
This Excel file presents a comprehensive healthcare data analysis project focused on deriving insights from patient and hospitalization data. It involves key steps such as data cleaning, transformation and integration of multiple datasets (medical examinations, hospitalization details and customer profiles). The file includes calculated fields like BMI-based weight status, diabetes classification and patient age, along with formatted healthcare charges. It further utilizes pivot tables, charts and an interactive dashboard to analyze relationships between factors such as age, health conditions and medical costs, enabling data-driven decision-making in the healthcare domain.
